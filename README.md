# RoV-SN Tournament Official

## Overview
RoV-SN Tournament Official is a web-based application designed to manage and display information for RoV (Arena of Valor) tournaments. The project provides features for viewing fixtures, results, team and player statistics, standings, and more, with both frontend and backend components.

## Features
- Tournament fixtures and results display
- Team and player statistics
- Standings and rankings
- Team and player profile pages
- Responsive web design
- Backend server for data management (Node.js)

## Project Structure
```
├── fixtures-results.html      # Fixtures and results page
├── format.html               # Tournament format information
├── index.html                # Home page
├── players.html              # Player list and details
├── script.js                 # Main JavaScript for frontend
├── standings.html            # Standings and rankings
├── statistics.html           # Player and team statistics
├── styles.css                # Main stylesheet
├── team-stats.html           # Team statistics
├── team.html                 # Team details
├── images/                   # Image assets (logos, visuals)
├── server/
│   ├── package.json          # Backend dependencies
│   └── server.js             # Node.js backend server
```

## Getting Started

### Prerequisites
- Node.js (for backend server)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/PhuriphatiZAMU/RoV-SN-Tournament-Official.git
   ```
2. Navigate to the server directory and install dependencies:
   ```bash
   cd RoV-SN-Tournament-Official/server
   npm install
   ```

### Running the Application
1. Start the backend server:
   ```bash
   node server.js
   ```
2. Open `index.html` in your web browser to view the frontend.
